## 🎯 **BooKnooK**

## 🚀 The Modern Way of Library "Study Room" Booking System

## 📑 OverView:

BooKnook is a full‑stack web application where students and library users can list study rooms they control (e.g., private rooms in a university library), and any registered user can browse, search, filter, and book those rooms for a specific date and time slot. The platform automatically prevents double‑booking using time‑conflict detection, allows room owners to manage their listings, and gives every user a dashboard to handle their own bookings. Secure JWT authentication is stored in HTTP‑only cookies, and the whole site is responsive and recruiter‑friendly.

## 🧩 key features of this project

🔐 1. Secure JWT Authentication

User registration and login with JWT stored in HTTP-only cookies for enhanced security.

🔍 2. Study Room Listing & Management

Authenticated users can create, update, and delete their own study room listings.

⏱️ 3.Time-Slot Based Room Booking

Users can reserve study rooms for specific dates and time slots through an intuitive booking system.

🟢 4. Personalized User Dashboard

Users can view, manage, and cancel their bookings from a dedicated dashboard.

🌐 5. Automatic Double-Booking Prevention

Smart time-conflict detection prevents overlapping reservations for the same room.

⚡ 6. Responsive & Modern UI

Built with Tailwind CSS ensuring a clean, fast, and mobile-friendly user experience.
